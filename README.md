# termux-miflash

### Installation
- apt update && apt upgrade
- apt install git
- apt install tar
- termux-setup-storage

- (extrak rom/firmware)
- cd /sdcard
- tar -xzvf /path/rom-file.tgz
- cd
```bash
git clone https://github.com/R1punk/termux-miflash
```
- cd termux-miflash
- bash install.sh
- bash fl.sh
